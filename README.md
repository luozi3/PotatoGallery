# WHY US?
1：用户端友好，部署后无需后端更改，一切修改在前端完成<BR>
2：针对二次元图片的现代检索系统，并且学习danbooru（手动打标/打标辅助）
3：流畅的前端动画与及其轻量的后台开销
4：完整的后台管理系统(注册开关，tag修改url指定)
5：高度稳定的服务器系统
