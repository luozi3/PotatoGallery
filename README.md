# PotatoGallery
适用于 0.5G RAM 服务器的高性能图片库。 UI 思路延续了Pixiv主页，标签参考了 Danbooru 的可编辑与打标，分类模式参考 Komga 设计。使用 Vue 3 和 SQLite FTS5 构建。<BR>
